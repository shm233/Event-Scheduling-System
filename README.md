# Event-Scheduling-System

<p align="center">Event Scheduling System
This project demonstrates a small Even Scheduling System that allows the user to log in and customize events, create new events, update existing events, view them and delete events when needed. Also this project allows user to see messages of what task he has committed to track records.</p>
